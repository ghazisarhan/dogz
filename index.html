Dogz is a Mobile Application that allow users to see and to receive, in real time, problems or outages happening on a Computer Network. It is composed by three separated entities, a broker, a middleware and the client application.

Dogz supports the following environments as backend for event and alerts generation:

  • Nagios
  • Naemon

Support for Zabbix and other Network Monitoring tools are also planned for near future and I hope to make it available as soon as possible

Let the dog walk you with this interface overview so you can see what the application looks like or see how to install and use Dogz

Features

  • Runs on Android and should run also on iOS and Windows(I have no phone to test it out)
  • Uses websockets, so, no delay when the client is open
  • Sends events through push notifications if client is closed
  • Performance graphs for all entities being monitored
  • Availability graph is also present
  • Filter events with one touch
  • Status view on a specific dashboard per host being monitored
  • Graph to show the overall health of the network
